ClawBox — 나만의 AI 에이전트, 탭 한 번으로

ClawBox는 클라우드에서 완전히 관리되는 AI 에이전트를 제공합니다 — 자체 컴퓨터, 브라우저, 메시징 기능을 갖추고 있습니다. API 키도, 서버도, 설정의 번거로움도 없습니다. 탭 한 번이면 시작됩니다.

항상 실행되는 나의 AI 에이전트
ClawBox는 단순한 챗봇이 아닙니다 — 자체 클라우드 컴퓨터를 가진 완전한 AI 에이전트입니다. 앱에서 직접 대화하거나, Telegram에 연결하거나, 실시간 화면 보기로 작업 과정을 지켜볼 수 있습니다.

원하는 방식으로 대화하세요
• 앱 안에서 AI 에이전트와 직접 대화 — 외부 도구 필요 없음
• Telegram 봇을 연결해 이미 사용 중인 메신저에서 대화
• 양쪽 모두에서 매끄러운 경험 — 에이전트가 모든 것을 기억합니다

AI를 위한 진짜 컴퓨터
에이전트는 풀 브라우저가 설치된 실제 클라우드 머신에서 실행됩니다. 웹을 탐색하고, 양식을 작성하고, 주제를 조사하고, 행동을 취할 수 있습니다 — 여러분이 하는 것처럼.
• 헤드풀 브라우징 — 에이전트가 실제 웹 페이지를 보고 상호작용합니다
• 실시간 VNC 화면 — 에이전트의 데스크톱을 실시간으로 확인

스킬 허브 — 에이전트를 더 강력하게
앱에서 직접 스킬을 탐색하고 설치하여 에이전트에게 새로운 능력을 부여하세요. 오픈 스킬 마켓플레이스 ClawHub가 제공합니다.
• 원탭 설치 — 에이전트에 새로운 기능을 즉시 추가
• 탐색 및 검색 — 이름이나 키워드로 스킬 검색
• 간편 관리 — 설치된 스킬 확인, 버전 체크, 원탭 제거
• 스킬 상세 정보 — 설치 전 작성자, 버전 이력, 변경 사항 확인
• 계속 성장 — ClawHub 커뮤니티에서 새로운 스킬이 추가됩니다

오픈클로 게이트웨이
에이전트는 안전한 OpenClaw Gateway를 통해 연결됩니다 — 암호화되어 항상 보호됩니다. 파워 유저는 Gateway 웹 콘솔에 접속하여 에이전트 설정을 세밀하게 조정하고, 구성을 관리하며, 완전한 제어권을 가질 수 있습니다.

설정 없이 바로 시작
• LLM API 키 관리 불필요 — 모든 것이 포함되어 있습니다
• 원탭 클라우드 인스턴스 생성
• 단계별 Telegram 봇 설정 마법사
• 몇 초 만에 에이전트가 활성화됩니다

완전 관리형 클라우드
AI 에이전트가 클라우드에서 24시간 실행됩니다. ClawBox가 배포, 업데이트, 모니터링을 처리합니다 — 여러분은 중요한 일에만 집중하세요.
